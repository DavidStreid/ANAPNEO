# ANAPNEO

## Description
An app-based rewards program to improve lung cancer prognosis.

## GETTING STARTED
~~~~
$ git clone https://github.com/DavidStreid/ANAPNEO.git
$ cd ANAPNEO
$ ./anapneo.sh
~~~~

Server should be started at http://localhost:4200/
